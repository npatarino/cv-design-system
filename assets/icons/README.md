## Generative AI Rationale
If you need to generate additional icons using Generative AI, use the following style rationale to ensure they perfectly match the existing design system aesthetic:

- **Style:** Minimalist, two-dimensional line art (outline only). No solid fills, gradients, drop shadows, or 3D perspective.
- **Stroke:** Use a consistent, unified stroke width (typically 1.5px or 2px). Ensure it relies on `currentColor` for theming rather than hardcoded colors.
- **Edges & Corners:** All lines must have rounded caps (`stroke-linecap="round"`) and rounded joins (`stroke-linejoin="round"`). Avoid sharp, pointy corners.
- **Geometry:** Simple, clean, and highly legible geometric shapes. Do not add unnecessary embellishments or overly intricate details.
- **Composition:** Centered within a square viewport (e.g., `0 0 24 24`), maintaining optical balance and proper breathing room (padding) around the edges.
